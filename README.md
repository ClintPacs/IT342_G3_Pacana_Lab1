## Quick Start

```bash
# 1. Start MySQL and create DB
# Run: CREATE DATABASE brewbatch_db;

# 2. Start Backend
cd backend && mvn spring-boot:run

# 3. Start Frontend (in a new terminal)
cd web && npm install && npm start
```

Visit `http://localhost:3000` to use the app.
